About Project

This is a project to buy products online,it contains 3 pages...


First page is the store page where you can see all products with its price and add to cart button, when you click this button you can see the red sign in the top right corner that shows the number of products in the cart was increased.


Cart page shows the detected products with total price and items, you can increase or decrease the quantity of each item.


if you are not registered you need to enter some information about you,otherwise you will see only one form.


checkout buton will take you to checkout page where you can get your products.


This page shows a summary and shipping form, You need this form if any of these products need to be shopped, It takes information like city,address,state and zipcode.
